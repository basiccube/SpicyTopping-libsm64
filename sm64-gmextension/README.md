This directory contains parts of the GM extension that is meant to be used alongside the fork.
The extension isn't complete and several functions + constants are missing due to it being meant for the specific case I used it for.  

The reason why I'm including it here this way is because of how much of a complete mess I made it for myself.
You're way better off just taking whatever is here and doing your own thing.

This is all heavily based on [headshot2017's libsm64-gm8 repo](https://github.com/headshot2017/libsm64-gm8), I just kinda threw it at current versions of GM and that's about it.  

Dependencies that you'll need:
* libpng
* SDL2
* pthreads

The example directory contains some GM objects and code. Just like how the extension is incomplete, this too is also meant for the very specific reason I used it for, so you're also on your own getting any of that to work.
